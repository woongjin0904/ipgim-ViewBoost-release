# 🚀 ViewBoost Release & Update Center

이 저장소는 **ViewBoost** 프로그램의 공식 배포 및 업데이트 관리를 위한 전용 공간입니다. 최신 버전의 설치 파일과 자동 업데이트를 위한 메타데이터를 제공합니다.

---

## 📥 초기 설치 방법 (Initial Installation)

처음 프로그램을 설치하시는 분은 아래 단계에 따라 최신 버전을 다운로드할 수 있습니다.

1. 오른쪽 사이드바의 **[Releases]** 섹션을 클릭하거나 [여기](https://www.google.com/search?q=https://github.com/woongjin0904/ipgim-ViewBoost-release/releases)를 눌러 최신 버전으로 이동합니다.
2. 해당 버전의 **Assets** 목록에서 실행 파일(예: `ViewBoost-Setup.exe` 또는 `.zip`)을 다운로드합니다.
3. 다운로드한 파일을 실행하여 설치를 완료합니다.

---

## 🔄 업데이트 관리 (Update Management)

ViewBoost는 이 저장소의 최신 릴리스 태그와 파일을 기반으로 업데이트를 확인합니다.

* **자동 업데이트:** 프로그램 실행 시 이 저장소의 최신 버전을 자동으로 감크하고, 필요 시 패치 파일을 다운로드하여 업데이트를 진행합니다.
* **수동 업데이트:** 새 기능이 추가되었을 때, 릴리스 노트를 확인하고 최신 Assets를 다시 다운로드하여 덮어쓰기 할 수 있습니다.
